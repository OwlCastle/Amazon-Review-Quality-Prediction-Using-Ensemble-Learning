# Amazon-Review-Quality-Prediction-Using-Ensemble-Learning

## Project Description

This project was developed as part of a Machine Learning course assignment.

The objective of this project is to predict the quality of Amazon product reviews using ensemble learning techniques.

The project aims to automatically identify high-quality and low-quality reviews based on textual and statistical features, helping improve the reliability of online review systems. It includes data preprocessing, feature engineering, model training, performance evaluation, and prediction using multiple ensemble learning algorithms.

## Dataset

The original dataset was provided by the course instructor for educational purposes.

Due to GitHub's file size limitations, the dataset is hosted separately on Hugging Face:

**https://huggingface.co/datasets/OwlCastle/Amazon-Review-Quality-Prediction-Using-Ensemble-Learning-dataset**

After downloading, extract the dataset into the project's `data/` directory.
